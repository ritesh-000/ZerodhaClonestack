# Zerodha Clone

A full-stack Zerodha Clone built using the MERN Stack. The project replicates the core UI and functionality of Zerodha, including a landing page, dashboard, holdings, positions, orders, and a basic signup page.

## 🚀 Live Demo

### Frontend
https://zerodha-frontend-4npd.onrender.com

### Dashboard
https://zerodha-dashboard-zv31.onrender.com




## Features

- Responsive Landing Page
- Dashboard
- Holdings
- Positions
- Orders
- Buy/Sell Window UI
- Stock Charts
- MongoDB Integration
- REST API using Express
- React Router Navigation



##Tech Stack

### Frontend
- React.js
- React Router
- Bootstrap
- Axios

### Dashboard
- React.js
- Material UI
- Chart.js
- React ChartJS 2

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
